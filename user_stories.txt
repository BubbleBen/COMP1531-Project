Epic 1: As a user of the Slackr, I should be able to log in, log out, be able to create a new account with a valid email address and reset the 
password if forgotten.
===================================================================================================================================================
1. As a new user to Slackr, I would like to be able to register with my email and set my own password, so that I can log in at any time and 
anywhere with the account that I have created.
2. As a user of the Slackr, I would like to be able to log out of my account, so that if I have logged in a public device, I could sign out and 
protect my personal information.
3. As a user of the Slackr, I would like to be able to reset my password through the email I registered with, so that if I accidentally forget my 
old password, I can still get my access for the same account back using my email.

Epic 2: As a user of the Slackr, I should be able to create, search, join and leave channels. I should be able to modify a user’s admin privileges.
===================================================================================================================================================
1. As a user of the Slackr, I would like to create public channels, so that I can get people interested in the same topic to join and I can see 
other people’s opinions and make more friends at the same time.
2. As a user of the Slackr, I would like to be able to create private channels, so that I can share things with only the people I know.
3. As a user of the Slackr, I would love to have the ability to search for all the existing channels, so that I can always check if there are new 
channels I am interested in and join it.
4. As a user of the Slackr, I would like to be able to see all the channels I have already joined, so that I can get a clear image of how many 
channels I have joined or if I want to get rid of some.
5. As a user of the Slackr, I would love to be able to invite people to my channels, so that I can create special discussion groups at any time I 
want. 
6. As a user of the Slackr, I would like to have the ability to leave channels, so that if I do not like the channel or I do not belong to the 
group of people in that specific channel anymore.
7. As an owner of a channel in Slackr, I would like to be able to modify a member's privilege, so that I can manage and organize my channel more 
efficiently. 

Epic 3: As a member of a channel in Slackr, I should be able to send, edit, remove, pin and react to messages. I should also be able to search for 
messages. I should be able to begin a “stand up” as well.
===================================================================================================================================================
1. As a member of a channel in Slackr, I would like to be able to send messages, so that people in the channel can communicate with each other by 
sending messages.
2. As a member of a channel in Slackr, I would like to be able to pin and unpin a message, so that I can mark the messages that I’m interested or I 
think it’s important, and also that I can unpin it when I don’t need the message anymore.
3. As a member of a channel of Slackr, I would like to be able to react and unreact to messages, so that I can express my emotion and opinion on a 
certain message faster and more straight forward, and if I don’t want my emotions and opinions to be seen, I can unreact the message I reacted to.
4. As a member of a channel of Slack, I would like to be able to edit a message, so that when I said something wrong I can still change the message 
even after it’s been sent already.
5. As a member of a channel of Slack, I would like to be able to remove a message, so that when I don’t want the message I sent to be seen anymore, 
I can delete it at any time I want.
6. As a member of a channel of Slack, I would like to be able to search for messages based on key words, so that I can access old content 
efficiently.
7. As a member of a channel of Slack, I would like to be able to begin a “stand up”, so that I can perform Agile stand-up method better in a group 
meeting, which means everyone in the group can get together talk for 15 minutes and make sure everyone knows what’s been done since the last 
meeting and what’s everyone’s plan before the next meeting. 

Epic 4: As a valid member of Slackr, I should be able to edit my own profile and I should be able to see other people’s profile as well.
===================================================================================================================================================
1. As a valid member of Slackr, I should be able to customise my personal handle, so that I can freely switch my “nickname” whenever I want and to 
whatever I want so that the account is more personal and more friendly to other members.
2. As a valid member of Slackr, I should be able to customise my personal details, so that if I changed, for example, names or email addresses, I 
would be able to update as soon as possible on Slackr as well.
3. As a valid member of Slackr, I should be able to check out other people's profiles as well, so that I can make sure if a person is actually the 
person I know by checking his or her profiles.

One more: As an admin of Slackr, I would like to be able to make another person an admin as well, so that I can ask other people to help me manage 
and organize Slackr application better.
